Template .Net 5 / Angular 11

Welcome to this template created to help the developers to start a new project using .Net as backend platform and Angular 11 as frontend framework.

Please, look below the links to help you to know more about all technologies in this project.

  • .Net Platform, ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
  • MySQL for database

The pre-requisites to run this template in your local machine are:

  • Visual Studio 2019 Community
  • .Net 5 SDK
  • Angular CLI 11
  • MySQL Developer

    • To help you get started, we've also set up:

    • Create Account: Find it in Login > Create Account. There is fields validations, e-mail notifications, class mappings, etc.
    • Authenticate / Change Password. Find it in Login Page. There is forms to remember password, validate code and change it. There is authorisation in client and server side.
    • Authorization and Permissions. Managed by the server. There is private API's to authorize (or not) if the user can access this data. Also, there is a middleware in client side to send a JWT token to validate the user logged.
    • Change Account Details. There is a default page to user logged change his name.
    • Admin Features. The admin user has different modules to access than normal user which he can see all the users registered on the platform.
    • Profile and Modules Management. This template has the functionality in the server side to manage modules by each profile. Find it in MySQLContext in Data project.
    • E-mail Notification. There is a micro service in CrossCutting layer to send the notifications by e-mail. You have to update the server e-mail information in appsettings.json
    • Swagger Documentation. There is a swagger documentation which will show all APIs documentation by the URL /api/documentation which is updated automatically when you create new APIs.
